# Pocket CHIP Menu

A simple PyGame replacement for the Pocket CHIP home launcher

Still WIP.

## Installation

```
cd ~
git clone https://github.com/omgmog/pocketchip-menu.git
cd pocketchip-menu
sudo apt install python3-pip
pip install -r requirements.txt
python3 main.py
```
